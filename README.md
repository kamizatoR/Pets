# Pets

## サイト概要
### サイトテーマ
ペットや動物を画像や数秒程度の動画で共有するSNS

### テーマを選んだ理由
某SNSを利用して動物（ペット）の動画・画像を閲覧している際に、他のコンテンツもおすすめされることが多く、不便に感じており、動物（ペット）が好きという層にもう少し絞ったサービスコンテンツがあればと思い着手いたしました。

本サービスでは、他ユーザーへの共有やアクション、自身だけの思い出や記録としても利用できるような機能（タイムラインやユーザー非公開機能）、また初めてのユーザーもすぐ利用できるゲストログインも用意しております。

### ターゲットユーザ
動物が好きな人・自身のペットを投稿し他のユーザーからのリアクション、または思い出として残したい人

### 主な利用シーン
動物を見て癒されたい時やペットを共有したい時など
